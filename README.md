Simple android application which include

1.Social Login wiht Google and Facebook both using firebase-ui auth.

2.It also fetches users details like username and email provided by 
  the selcted provider, in this case we have only two provider Facebook 
  and Google.

3.User details is stored in firebase realtime database useing unique
  key provided by the firebase to each authenticated user.

4.It have login and logout options and mentais the sessions very well.
